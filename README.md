# MetrobusCDMX
Library to write readable and reproductible pipelines using decorators and generators.
Tested for Python > 3.6.9.

## Tools And Frameworks usage

- python3
- flask
- flask-sqlalchemy
- shapely
- pandas
- PostgreSQL
- docker
- docker-compose

## Init Containers

Linux: 
`chmod +x setup.sh && sh start.sh`

Windows:
`.\start.bat`

## Steps

- processing data
- check/transform
- orm-model
- orm-model -> resources
